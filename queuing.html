QUEUE RESERVATION SYSTEM OF MEDICAL CENTRE KMITL

Web application (Django)

Web application queue reservation system of medical centre KMITL is web booking queue for KMITL medical center, it is a project created in web programming subjects in software engineering course , by grouping of 3 people. I was responsible for front-end , another 2 people were responsible for back-end.

Normally, KMITL medical center does not have to book an queue through the website to make an appointment. Currently, it will be booking a manual queue, which we see the current problem is that some patients forget to an appointments, lost an appointments paper or booking queue but wait so long time cause skip them queue . That's reason we created an appointment booking website for KMITL medical center.
